# CASH CHECK 배포용 - 회원가입 없음

Supabase Auth 회원가입/로그인 없이 바로 설문을 둘러보고 참여 링크로 이동하는 배포용 복제본입니다. 원본 `설문조사 MVP` 폴더는 그대로 유지합니다.

- `index.html`: 경품을 먼저 고르는 랜딩페이지
- `surveys.html`: 설문 추천, 검색, 상세 정보를 제공하는 설문페이지
- `survey-data.js`: 랜딩페이지와 설문페이지가 함께 사용하는 설문 데이터
- `app.js`: 설문 목록, 상세 모달, 필터, 더 보기 동작
- `landing.js`: 경품 카드 랜딩 동작

## 포함된 기능

- 경품을 먼저 보고 설문을 고르는 한 화면 오프닝
- 접속할 때마다 달라지는 무작위 추천 설문
- 카테고리 필터와 키워드 검색
- 설문 상세 정보 및 개인정보 수집 범위 확인 모달
- 설문 시작 버튼에서 외부 설문 링크로 바로 이동
- 반응형 모바일 메뉴와 접근성 기본 처리
- CASH CHECK 직접 추첨 원칙 안내

## 제외된 기능

- 회원가입
- 로그인
- Supabase Auth 연결
- 계정별 중복 참여 기록 저장

## 설문 데이터 수정 방법

현재 설문 목록은 `survey-data.js`의 `window.MOAFORM_SURVEYS` 배열로 관리합니다. 새 설문을 전달받으면 설문명, 설명, 카테고리, 예상 시간, 추첨 경품, 모집 인원, 마감일, 수집 정보, 등록 기관, 실제 설문 링크를 이 파일에 추가하면 됩니다.
